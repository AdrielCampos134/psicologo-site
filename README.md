# Site Psicologia - Dra. Thayline Silva Menezes

Site profissional para psicóloga clínica com design premium e funcionalidades modernas.

## 🎨 Design e Funcionalidades

- **Design Premium**: Paleta de cores sofisticada com azul profundo e índigo
- **Interface Responsiva**: Adaptada para desktop, tablet e mobile
- **Formulário WhatsApp**: Integração direta com WhatsApp para contato
- **Animações Suaves**: Transições e efeitos hover elegantes
- **Cards Profissionais**: Layout moderno com sombras sutis

## 🛠️ Tecnologias Utilizadas

- **Frontend**: HTML5, CSS3, Tailwind CSS, JavaScript Vanilla
- **Backend**: Node.js com Express
- **Ícones**: Font Awesome
- **Fontes**: Google Fonts (Inter)

## 📁 Estrutura do Projeto

```
site-psicologia/
├── public/
│   ├── index.html          # Página principal
│   └── script.js          # Funcionalidades JavaScript
├── server.js              # Servidor Node.js
├── package.json           # Dependências do projeto
├── .env                 # Variáveis de ambiente
├── .gitignore           # Arquivos ignorados pelo Git
└── README.md            # Documentação
```

## 🚀 Como Executar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor:
   ```bash
   npm start
   ```

3. Acesse o site em: `http://localhost:3000`

## 🎯 Características Principais

### Design Premium
- Paleta de cores profissional (azul profundo, índigo, cinzas)
- Tipografia Inter com hierarquia clara
- Cards com bordas sutis e sombras elegantes
- Hero section com gradiente profissional

### Funcionalidades
- Menu responsivo com animação
- Scroll suave entre seções
- Formulário de contato via WhatsApp
- Formatação automática de telefone
- Animações de entrada ao scroll
- Estados de loading nos formulários

### Seções
- **Home**: Hero section com CTA principal
- **Sobre Mim**: Apresentação profissional com foto
- **Especialidades**: Cards para Individual, Casal e Infantil
- **Como Funciona**: Processo em 4 passos
- **Depoimentos**: Feedback de clientes
- **Informações Práticas**: Detalhes de atendimento
- **Contato**: Formulário WhatsApp integrado
